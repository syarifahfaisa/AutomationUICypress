# AutomationUICypress
Preparation:
- make sure you have node.js and npm installed
  - if not, you can download it from here https://nodejs.org/en/download/prebuilt-installer
- install cypress from node:
  -  change the directory to your project path
  -  npm init
  -  npm install cypress

How to run:
  - npx cypress open
  - select browser
  - select file to run
